# 📚 Smart Study Planner

A modern, feature-rich **Flutter mobile application** designed to help students efficiently manage their academic life. From organizing subjects and tasks to tracking study progress, this app serves as an all-in-one productivity tool.

---

## 🚀 Overview

Smart Study Planner is built to simplify study management through a clean interface and structured workflow.

It allows users to:
- Plan their study schedules  
- Organize academic tasks  
- Track progress effectively  

Whether managing daily tasks or long-term goals, the app helps maintain consistency and productivity.

---

## ✨ Key Features

### 🔐 Authentication System
- Secure Login & Sign Up  
- Forgot Password functionality  
- User Profile Management (Edit Profile)  

### 📊 Dashboard
- Daily tasks overview  
- Subjects summary  
- Study progress insights  
- Quick navigation to features  

### 📘 Subject Management
- Add and manage subjects  
- Organize subjects efficiently  
- View subject-specific details  

### ✅ Task Management
- Create, edit, and delete tasks  
- Mark tasks as completed  
- Categorize tasks by subjects  
- Search and filter tasks  

### 🗓️ Study Planning
- Weekly study timetable  
- Structured scheduling  
- Organized academic planning  

### 📈 Progress Tracking
- Monitor study performance  
- Compare completed vs pending tasks  
- Visual productivity insights  

### 🔔 Notifications & Reminders
- Timely reminders  
- Stay updated with tasks and schedules  

### ⚙️ Settings
- Manage app preferences  
- Customize user experience

## 🛠️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Shaista45/study_planner_app.git

# Navigate into the project
cd smart_study_planner

# Install dependencies
flutter pub get

# Run the app
flutter run
```
